💻 **Projects**

**[AI Mancala](https://github.com/Abhi6310/AI_Mancala)** `Next.js · FastAPI · Redis · Supabase`
Deployed browser-playable Mancala with four AI opponents scaling from random play to alpha-beta pruning at search depth 10. Live session state managed through Redis; player history persisted in Supabase across a stateless API backend.

**[NLP Fine-Tuning Pipeline](#)** `Python · TinyBERT · TinyLLM · Hugging Face`
End-to-end pipeline for fine-tuning small language models on domain-specific text, with metrics logged at every training stage. Modular ingestion and encoding stages let the pipeline retarget different datasets or models without touching core logic.

**[NeuroTune](https://github.com/Abhi6310/NeuroTune)** `Python · React · Next.js · Local LLMs`
Real-time audio modulation platform where all inference runs on-device with no cloud dependency. Async pipelines between a React frontend and Python backend keep latency low enough for real-time use on consumer hardware.

**[STM32 Battleship](https://github.com/Abhi6310/STM32_Battleship)** `C · STM32 · LCD · Embedded Firmware`
Fully playable Battleship on an STM32 microcontroller with live board rendering on an LCD display and an AI opponent, written in bare-metal C with no OS. Game logic, display, and input all share the same processor cycles within tight memory constraints.
