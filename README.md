## Projects

**[SCORE](https://github.com/Abhi6310/SCORE)** `Python · TypeScript · FastAPI · Hugging Face · Next.js · Tone.js`

- A Schema-Constrained Output Retrieval Engine (SCORE) that turns a natural-language focus intent into a custom audio session, generated by a fully on-device language model with zero cloud calls
- Designed the pipeline to classify the request with a sentence-transformer, bound the model's JSON output with constrained decoding, and route any invalid result to a per-intent fallback valid by construction
- Engineered a retrieval feedback loop that biases each new session with the user's highest-rated past sessions, served through a FastAPI and WebSocket backend to a Next.js and Tone.js browser client
